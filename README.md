# Orange Cat Tetris 🐈

A browser-based Tetris clone with an orange-cat pixel style and built-in 8-bit retro music.

## Run it

Because this project is plain HTML/CSS/JS, you can run it with any static server.

```bash
python3 -m http.server 8000
```

Then open: <http://localhost:8000>

## Controls

- **Left/Right arrows**: Move
- **Down arrow**: Soft drop
- **Up arrow**: Rotate
- **Space**: Hard drop
- **P**: Pause/Resume
- **Music button**: Mute/Unmute soundtrack
- **Restart button**: Start a new game
